# 📻 Radio Net Record Tool (业余无线电台网签到记录工具)

这是一款专为业余无线电台网（特别是类似深圳梧桐山/439.4625MHz业余中继台网）设计的签到记录管理套件，界面采用了硬核沉浸式的类似 ICOM / FT-710 风格。

## ✨ 核心特性

- **高效快捷录入**：支持快捷回车键（Enter）全局一键通录入，最大化避免繁琐的鼠标操作，不打断主控思路。
- **智能记忆补全**：基于历史签到档案库（History Database），当您输入完整的呼号时，系统会自动补全这位老友常驻的 QTH（位置）、Power（功率）、Rig（设备）以及 Antenna（天线）等信息。
- **强制大写规范**：输入呼号（Callsign）时实时自动转为全部大写，严格符合业余无线电记录规范和习惯，并保持表格统一美观。
- **沉浸式暗场主题**：全局界面统一样式，去除了会让眼睛感到疲劳的交替斑马纹配色，保持统一的暗场黑客/无线电硬核风格。
- **多阶段主控切换支持**：在一场长达数小时的台网活动中遇到交接主控时，可快速插入带有醒目高亮底色的“阶段主控”提示分隔行，并在表头自动将新主控合并到“当值主控”合集中。
- **多端输出与备份**：自带防掉电和崩溃的自动归档机制；同时支持一键导出 **CSV数据表格**（存档或统计）以及 **PNG全量长截图**（用于微信群/公众号发榜单）。

## 🚀 使用说明与操作流程

### 1. 启动与基础台网信息确认
- 开启程序后，系统会自动读取本地最近的历史台站库以及备份日志作为防掉电保护。
- 在顶部的**台网信息面板**中分别设置：本场台网的**名称**、第一任**主控呼号**、**日期**和**预定时间**。这些信息不仅用于界面的横幅显示，也将在最后导出图片、表格时作为表头被一并输出。

### 2. 日常签到流水流程
遇到呼叫电台进来签到时，核心流程如下：
1. 在 **Callsign (呼号)** 栏输入对方呼号（程序会自动将输入强制转换为全部大写）。
2. 如果此前对方在此系统留有过历史记录，系统会在您继续输入或鼠标失去焦点时**自动通过记忆库补全**对方过去的信息（设备、天线、QTH等）。
3. 核对对方当次上报的信息。如果对方变更了位置(QTH)或者临时外场换了设备/天线，可以在对应栏直接修改。
4. 补充当前您实际给出的 **RST (信号报告)**（系统提供默认值 `59`）。如果需要可同时补充 **Topic (话题)** 或 **Remarks (备注)** 。
5. 按下键盘上的 **Enter/Return** 键 或点击右侧的 **"添加记录"** 按钮完成该笔数据入库。
6. 归档立刻生效，随后的光标和焦点会自动为您**瞬移回呼号输入框**，无缝衔接并等待下一个台网友台来报。

### 3. 主控交接 / 插入阶段特殊说明
如果是多段不同控制的台网，又或者有特邀主控进场：
1. 请先在左侧的 **Callsign (呼号)** 栏预先输入将要接棒的新主控呼号。
2. 点击金灿灿的 **"插入主控记录"** 按钮。
3. 弹出对话框提示您编写本阶段的阶段名或阶段说明（例如填入："下半场" 或 "特邀主控时段"）。
4. 操作完成后，目前数据大表层将插入一条高亮且加粗呈现的“隔断带（行）”，以隔开上半场与下半场的记录列表，并且自动收集该主控至表头的"所有主控人员名单中"。

### 4. 收尾导出与安全备份
- **防掉电备份机制**：即便意外遭遇电脑死机或断电跳闸中断，这几小时聊的天也不会白费，本地临时日志表都会记录着这笔入库信息。
- **导出 CSV 表单**：活动圆满落幕后，可点击 **导出 CSV** ，将该场次所有的抄收历史整理成表格，发送给中继群管理进行备份或分析。
- **导出高清长图**：如果你习惯在微信群每天群发“签到花名册大榜”，只需点击 **导出长图(PNG)** 等待几秒后即可生成一张精美的高清榜单图片进行转发。
